## Deskripsi
Askalot merupakan aplikasi pertanyaan-jawaban yang memungkinkan penggunanya berkomunikasi dan mencari informasi. 

Aplikasi ini menggunakan Supabase dalam implementasi backend, alasannya karena dengan Supabase file gambar dapat dimasukkan ke dalam storage secara gratis, sedangkan firebase membutuhkan biaya tambahan

## Fitur-fitur
- Sign up dan sign in menggunakan email untuk masuk ke akun
- Membuat pertanyaan (Ask) dan menjawab (Reply) pertanyaan orang lain, dan dapat menambahkan gambar untuk memperjelas pertanyaan.
- Upvote atau downvote jawaban atau opini yang diberikan oleh orang lain. Upvote artinya mempopulerkan postingan dan downvote sebaliknya.
- Pengguna dapat mengedit profil seperti foto profil, username, dsb
- Pengguna dapat mencari pertanyaan yang sesuai dengan yang diinginkan pengguna

## Instalasi

### A. Unduh File APK

1. Buka halaman repository GitHub.
2. Masuk ke tab Releases (jika tersedia), atau buka folder yang berisi file APK.
3. Klik file dengan ekstensi .apk.
4. Tekan tombol Download untuk mengunduh APK ke perangkat Android.

### B. Aktifkan Instalasi dari Sumber Tidak Dikenal

1. Buka Pengaturan (Settings) Android.
2. Masuk ke Keamanan atau Privasi.
3. Aktifkan Instal aplikasi tidak dikenal / Unknown Sources.
4. Pilih browser atau file manager yang digunakan untuk membuka APK.
5. Aktifkan Izinkan dari sumber ini

### C. Instal APK

1. Buka File Manager di Android.
2. Cari file APK yang sudah diunduh.
3. Ketuk file APK tersebut.
4. Tekan Install dan tunggu hingga proses selesai.
